## 概述
基于ARM+Linux体系构建的VMI（Virtual Mobile Infrastructure）云平台
## 架构
![OPENVMI架构](docs/img/architecture.png)
### 硬件支持
* 鲲鹏916  
* 鲲鹏920  
* FT2000+/64（仅支持64位APP运行）
### 支持的系统版本
* Ubuntu 18.04
* Ubuntu 20.04
